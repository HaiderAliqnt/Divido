DIVIDO — GIK'S own Campus Expense Splitter

DIVIDO is a campus-friendly expense sharing app, inspired by Splitwise, designed specifically for GIK students living in hostels.

It helps you track, split, and settle expenses effortlessly with your friends — whether it’s mess bills, food orders, or shared rides.

Features
-> Group Expense Management
    Create groups (roommates, project teams, trips)
    Add members easily
    Track shared expenses in one place
-> Smart Expense Splitting
    Split equally or unevenly
    Assign exact amounts per person
    Automatic balance calculation
-> Simplified Tracking
    See who owes whom
    Real-time balance updates
    Clean and minimal UI for quick usage
-> Transaction History
    View all past expenses
    Track payment records
    Transparent expense logs
-> Settlement System
    Mark payments as settled
    Reduce unnecessary transactions using net balances

Why DIVIDO?

Unlike generic apps, DIVIDO is built for GIKI life:

 -> Optimized according to campus menu
 -> Perfect for frequent small group expenses
 -> Fast and lightweight
 -> Built for trust within student communities

Tech Stack
-> Frontend: REACT
-> Backend: EXPRESS JS , NODE JS
-> Database: POSTGRESQL
-> Authentication: JWT
